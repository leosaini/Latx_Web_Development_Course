HTML - stands for Hypertext Markup Language
1. HTML use to create the structure and layout of website.
2. HTML use elements, sunch as heading, paragraph.
3. HTML is the foundation of any web page.
4. we use css and js also to create UI and UX better.

Markup Language : any language use tag to create web page structure
Hypertext : 
It know as a link.
its clickable elements in webpage that allow user to navigate other pages.


Tags : tags are markup language. 
       ex : <h1>Content</h1>

Attributes : provide extra inforamtion about tag


Basic Structure of HTML
<!DOCTYPE html>
<html>
    <head>
        <title>Document</title>
    </head>
    <body>       
        <!-- main content of web page -->
    </body>
</html>
